Detailed Description

Process mappings and apply them to the scene.

These functions process the mappings from the mappings file. In particular, they supplement <a https://github.com/DavidBrainard/RenderToolbox3/wiki/Generic-Scene-Elements">generic scene elements with default property values and convert them to renderer-native formats.

Users should not use these functions directly. Just use the BatchRender() function.

Apply Mitsuba mappings objects to the adjustments DOM.

Parameters
idMap
objects

Modify the document represented by the given idMap, with the given mappings objects. objects must be a struct array of mappings objects as returned from MappingsToObjects() or GenericObjectsToMitsuba().

Used internally by BatchRender().

Usage: ApplyMitsubaObjects(idMap, objects)

References CreateElementChild(), PrintPathPart(), SearchScene(), SetSceneValue(), and StringToVector().

objects GenericObjectsToMitsuba ( objects  )

Translate generic objects names and values to Mitsuba.

Parameters
objects

Convert the generic mappings objects to Mitsuba-native mappings objects. The conversion can change object class and subclass names, and property names, types, and values. The conversion may also add hints to be used by ApplyMitsubaObjects(). Returns the updates struct array of mappings objects.

Used internally by BatchRender().

Usage: objects = GenericObjectsToMitsuba(objects)

objects GenericObjectsToPBRT ( objects  )

Translate generic objects names and values to PBRT.

Parameters
objects

Convert the generic mappings objects to PBRT-native mappings objects. The conversion can change object class and subclass names, and property names, types, and values. The conversion may also add hints to be used by ApplyPBRTObjects(). Returns the updates struct array of mappings objects.

Used internally by BatchRender().

Usage: objects = GenericObjectsToPBRT(objects)

References StringToVector(), and VectorToString().

objects MappingsToObjects ( mappings  )

Group mappings into meaningful objects.

Parameters
mappings

Create a struct array of mappings objects, based on the struct array of mappings data as returned from ParseMappings(). Each mappings object incorporates a declaration and property configurations, taken mappings that use the same id, inside the same block. Returns a new struct array of mappings objects.

Used internally by BatchRender().

Usage: objects = MappingsToObjects(mappings)

objects SupplementGenericObjects ( objects  )

Fill in generic objects with default properties, if needed.

Parameters
objects

Fill in a struct array of mappings objects with additional default propertues. This is necessary for generic mappings objects, because different renderers have different native defaults. Properties that are already specified are left as-is. Returns the updated struct array of mappings objects.

Used internally by BatchRender().

Usage: objects = SupplementGenericObjects(objects)
