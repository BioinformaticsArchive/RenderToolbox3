Detailed Description

Convert Collada scenes to PBRT.

Theses functions make up the RenderToolbox3 Collada to PBRT scene file converter. Users should not use most of these functions directly. Just use the ColladaToPBRT() function.

Print a standard PBRT scene file statement to a file.

Parameters
fid
identifier
type
params

Write a standard PBRT statement with the given identifier, type, and struct array of params. The given fid should be a file descriptor as returned from fopen().

identifier should be a string PBRT identifier, like 'Film', 'Material', 'Shape', 'etc. type should be a string PBRT object type that agrees with the identifier, like 'image', 'plastic', or 'trianglemesh'.

params should be a struct array with one element per parameter to be listed. It should have the following fields:

  • name - the name of a PBRT object parameter like 'xresolution', 'roughness', or 'P'
  • type - a PBRT value type, like 'float', 'point', or 'string'
  • value - a value to use for the named parameter. numeric values will be converted to strings.

The resulting PBRT scene file statement will look something like this: identifier "type" "params(1).type params(1).name" [params(1).value] "params(2).type params(2).name" "params(2).value" etc...

Used internally by ColladaToPBRT().

Usage: PrintPBRTStatement(fid, identifier, type, params)
